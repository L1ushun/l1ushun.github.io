Next 中的懒加载

React 提供了 lazy + Suspense 实现了懒加载。在 Next 中额外提供了 next/dynamic 方式实现懒加载。

使用时需要注意的是

  1. import 中不能是模板字符串和变量
  2. dynamic 要在模块顶层

客户端组件默认会被预渲染（SSR），通过 ssr: false 可以关闭预渲染，这会让组件比正常预渲染的组件晚显示，实际上会有一个 template 给该组件占位，然后等待加载完成再渲染。
