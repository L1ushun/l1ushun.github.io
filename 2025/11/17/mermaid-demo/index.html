Mermaid图表示例

Mermaid图表示例

本文展示了如何在Hexo博客中使用Mermaid绘制各种图表。

流程图

graph TD
    A[开始] --> B{判断条件}
    B -->|条件1| C[处理1]
    B -->|条件2| D[处理2]
    C --> E[结束]
    D --> E

时序图

sequenceDiagram
    participant 浏览器
    participant 服务器
    浏览器->>服务器: 请求数据
    服务器-->>浏览器: 返回数据
    浏览器->>浏览器: 渲染页面

甘特图

gantt
    title 项目计划
    dateFormat  YYYY-MM-DD
    section 阶段1
    需求分析     :a1, 2025-11-01, 7d
    设计规划     :a2, after a1, 5d
    section 阶段2
    编码实现     :a3, after a2, 10d
    测试验证     :a4, after a3, 5d
    section 阶段3
    部署上线     :a5, after a4, 2d

类图

classDiagram
    class Animal {
        +String name
        +int age
        +makeSound()
    }
    class Dog {
        +fetch()
    }
    class Cat {
        +climb()
    }
    Animal <|-- Dog
    Animal <|-- Cat

状态图

stateDiagram-v2
    [*] --> 待处理
    待处理 --> 处理中: 开始处理
    处理中 --> 已完成: 处理完成
    处理中 --> 异常: 出现错误
    异常 --> 处理中: 重试
    已完成 --> [*]

饼图

pie title 技术栈占比
    "前端" : 40
    "后端" : 30
    "数据库" : 20
    "运维" : 10

用户旅程图

journey
    title 用户购买流程
    section 浏览阶段
      访问网站: 5: 用户
      浏览商品: 4: 用户
    section 购买阶段
      加入购物车: 3: 用户
      结算: 2: 用户, 系统
    section 支付阶段
      选择支付方式: 3: 用户
      完成支付: 5: 用户, 系统

使用说明

要在文章中使用Mermaid图表，需要：

  1. 在文章Front-matter中添加mermaid: true
  2. 使用代码块语法，指定语言为mermaid

更多语法和示例可以参考[Mermaid官方文档](https://mermaid-js.github.io/mermaid/)。
